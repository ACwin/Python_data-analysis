### 通过分析某宝中秋月饼的销售情况，看看哪些口味月饼卖得好，哪些地方月饼卖得好

**涉及到的库：**
- Pandas — 数据处理
- Pyecharts — 数据可视化
- jieba — 分词
- collections — 数据统计

可视化部分：
- Bar — 柱状图
- Pie — 饼状图
- Grid— 分布
- Map— 地图
- Page — 页面插件
- Stylecloud — 词云图
